What is Auto Submit Site Domain?

Auto Submit Site Domain is a free, open-source tool that helps small businesses and indie developers improve their website visibility in Google Search. It automatically submits your websites to Google's index using either the "site:" operator method or Google Search Console API, ensuring your content gets discovered faster.

How does the tool work?

Our tool works in three simple steps:

  1. Domain Collection: It uses the Cloudflare API to fetch all domains and subdomains in your account.
  2. URL Extraction: It parses sitemaps for each domain to extract all URLs. If no sitemap is found, it extracts links from the homepage.
  3. Google Submission: It submits each URL to Google using either the "site:" operator through browser automation or direct submission via Google Search Console API.
What's the difference between the "site:" operator method and GSC API?

The "site:" operator method uses browser automation to search for your URLs in Google, which triggers Google's crawling and indexing process without any daily limits. The Google Search Console (GSC) API method submits URLs directly to Google's indexing API but has strict daily limits (typically around 200 URLs per day). Our tool uniquely offers both methods, giving you the best of both worlds.

Do I need a Google account to use this tool?

No, you don't need a Google account when using our "site:" operator method. This is a significant advantage over competitors who exclusively rely on Google Search Console API, which requires Google account verification. However, if you choose to use the optional GSC API integration, you will need to have your site verified in Google Search Console.

How long does it take for my pages to get indexed?

While we can't guarantee specific timeframes as Google's indexing process is ultimately controlled by Google, our users typically see their pages get indexed much faster than with traditional methods. Some pages may be indexed within hours, while others might take a few days. The tool significantly reduces the waiting time compared to passive methods where you simply wait for Google to discover your content.

Is this tool really free?

Yes, the core functionality of our tool is completely free and open-source. You can use the "site:" operator method to submit unlimited URLs without any cost. We also offer premium plans with additional features like advanced GSC integration, domain filtering options, and enterprise-scale support for businesses with more complex needs.

How do I set up the tool?

Setting up the tool is straightforward:

  1. Fork our GitHub repository
  2. Configure your Cloudflare API credentials (or use another domain source)
  3. Set up the GitHub Action to run on a schedule or trigger it manually
  4. Optionally, configure Google Search Console API access if you want to use that method

Detailed setup instructions are available in our documentation.

Can I use this tool for multiple websites?

Absolutely! Our tool is designed to handle multiple websites efficiently. When using the Cloudflare integration, it automatically fetches all domains and subdomains in your account. You can also configure domain allowlists to specify exactly which domains you want to submit. This makes it perfect for agencies managing multiple client websites or businesses with several web properties.
